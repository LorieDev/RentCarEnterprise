# RentCarEnterprise 

This is a full-stack car rental application I built while learning Java, DevOps, and Cloud technologies.

Live App:  
http://18.226.8.88:8080/index.html

---

## Project Purpose

This project started as a simple Java OOP practice and evolved into a complete real-world application deployed on AWS.

The goal was to understand how everything connects:

Java → Backend → Database → Frontend → DevOps → Cloud

---

## What I Built & Learned

### Java (OOP)
- Created classes and objects
- Used inheritance and polymorphism
- Built a clean object-oriented structure

### Spring Boot (Backend)
- Built REST API endpoints
- Connected backend to frontend
- Managed business logic

### MySQL (Database)
- Stored car data
- Managed availability and updates
- Connected using JPA (Hibernate)

### Frontend (HTML, CSS, JavaScript)
- Created a simple UI
- Allowed users to:
  - Add cars
  - View cars
  - Interact with backend API

### JUnit Testing
- Wrote tests to validate logic
- Ensured application works correctly

### Maven
- Managed dependencies
- Built the project into a JAR file
